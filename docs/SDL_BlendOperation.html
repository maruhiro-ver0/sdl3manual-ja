SDL_BlendOperation

ピクセルの要素の混合で使用するブレンド操作の列挙体

ヘッダ

SDL3/SDL_blendmode.h

構文

typedef Uint32 SDL_BlendOperation;

値

SDL_BLENDOPERATION_ADD0x1dst + src : 全てのレンダラが対応
SDL_BLENDOPERATION_SUBTRACT0x2src - dst : D3D, OpenGL, OpenGLES, Vulkanが対応
SDL_BLENDOPERATION_REV_SUBTRACT0x3dst - src : D3D, OpenGL, OpenGLES, Vulkanが対応
SDL_BLENDOPERATION_MINIMUM0x4min(dst, src) : D3D, OpenGL, OpenGLES, Vulkanが対応
SDL_BLENDOPERATION_MAXIMUM0x5max(dst, src) : D3D, OpenGL, OpenGLES, Vulkanが対応

バージョン

SDL 3.2.0以降
