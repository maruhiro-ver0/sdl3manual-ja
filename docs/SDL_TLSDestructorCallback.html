SDL_TLSDestructorCallback

データの後処理としてSDL_SetTLSに渡すコールバックの型

ヘッダ

SDL3/SDL_thread.h

構文

typedef void (SDLCALL *SDL_TLSDestructorCallback)(void *value);

引数

void *valueSDL_SetTLSに渡したポインタ

詳細

これはスレッドが終了したとき呼ばれるため, ここでアプリケーションの資源を解放することができる.

バージョン

SDL 3.2.0以降
