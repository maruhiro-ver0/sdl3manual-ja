SDL_SetJoystickVirtualHat

オープンした仮想ジョイスティックのハットの状態を設定する

ヘッダ

SDL3/SDL_joystick.h

構文

bool SDL_SetJoystickVirtualHat(SDL_Joystick *joystick, int hat, Uint8 value);

引数

SDL_Joystick *joystick状態を設定する仮想ジョイスティック
inthat更新する仮想ジョイスティックのハットのインデックス番号
Uint8valueハットの新しい値

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

ここで設定した値は, 直接または(SDL_PollEvent, SDL_PumpEvents, SDL_WaitEventTimeout, SDL_WaitEvent以外を含む)様々なSDL APIを通してSDL_UpdateJoysticksが呼ばれるまで適用されない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
