SDL_ReadSurfacePixelFloat

サーフェイスにピクセルを書き込む

ヘッダ

SDL3/SDL_surface.h

構文

bool SDL_WriteSurfacePixelFloat(SDL_Surface *surface, int x, int y, float r, float g, float b, float a);

引数

SDL_Surface *surface書き込み先サーフェイス
intx水平方向の位置. 0 <= x < 幅
inty垂直方向の位置. 0 <= y < 高さ
float *r通常は0～1の範囲の赤要素
float *g通常は0～1の範囲の緑要素
float *b通常は0～1の範囲の青要素
float *a通常は0～1の範囲のα要素

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数は速度よりも正確さを優先している. 単体テストには適しているが, ゲームエンジンでの使用を想定していない.

スレッドセーフ

この関数は別のスレッドから別のサーフェイスに対して呼ぶことができる.

バージョン

SDL 3.2.0以降
