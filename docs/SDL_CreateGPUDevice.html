SDL_CreateGPUDevice

GPUコンテキストを生成する

ヘッダ

SDL3/SDL_gpu.h

構文

SDL_GPUDevice * SDL_CreateGPUDevice(SDL_GPUShaderFormat format_flags, bool debug_mode, const char *name);

引数

SDL_GPUShaderFormatformat_flagsアプリケーションが提供できるシェーダの形式のビットフラグ
booldebug_modeデバッグモードのプロパティと検証の有効状態
const char *name優先するGPUドライバ. NULLのときSDLが最適なドライバを選択する

戻り値

(SDL_GPUDevice *) 成功のときGPUコンテキストを戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

GPUドライバ名は次の内の1つである:

  • "vulkan": Vulkan
  • "direct3d12": D3D12
  • "metal": Metal
  • NULL: SDLに最適なドライバを選択させる

バージョン

SDL 3.2.0以降
