SDL_fabs

xの絶対値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_fabs(double x);

引数

doublex浮動小数点の値

戻り値

(double) xの絶対値を戻す.

詳細

定義域: -INF <= x <= INF

値域: 0 <= y <= INF

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_fabsfを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
