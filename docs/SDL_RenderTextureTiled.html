SDL_RenderTextureTiled

テクスチャの一部を現在のレンダーターゲットにサブピクセルの精度でコピーする

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_RenderTextureTiled(SDL_Renderer *renderer, SDL_Texture *texture, const SDL_FRect *srcrect, float scale, const SDL_FRect *dstrect);

引数

SDL_Renderer *rendererテクスチャの一部がコピーされるレンダラ
SDL_Texture *textureコピー元のテクスチャ
const SDL_FRect *srcrectコピー元の領域. NULLのときテクスチャ全体
floatscalesrcrectのコピー先への転送時の拡大率. 例えば32×32テクスチャでscaleが2ならば64×64をタイル状に敷き詰める
const SDL_FRect *dstrectコピー先の領域. NULLのときレンダーターゲット全体

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

srcrectのピクセルはdstrectを敷き詰めるように繰り返す.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
