SDL_RunApp

SDLアプリケーションの初期化と起動を行う. mainFunctionを呼ぶ前に環境固有の初期化を行い, mainFunctionから戻った後に後処理を行う. これが必要なのは特定の環境で必要で, そうでない場合はただmainFunctionを呼ぶだけである.

ヘッダ

SDL3/SDL_main.h

構文

int SDL_RunApp(int argc, char *argv[], SDL_main_func mainFunction, void *reserved);

引数

intargcANSI-Cスタイルのmain関数のargc
char **argvANSI-Cスタイルのmain関数のargv
SDL_main_funcmainFunctionSDLアプリケーションのCスタイルのmain(). この関数の呼び出し元ではない. 名前は何でもよく, 実際のmainである必要はない
void *reserved必ずNULLにすること(将来のための予約. 環境固有の設定になるかもしれない)

戻り値

(int) mainFunctionが戻した値を戻す: 成功のとき0, その他のとき異常. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数は, SDL_mainではなく自分自身のmain()を使いたい場合(SDL_MAIN_HANDLEDを使用した場合など)に使用する. この関数を使う場合は, SDL_SetMainReady()は必要ない.

スレッドセーフ

通常はこの関数は初期化処理スレッドからスタートアップ時に1度だけ呼ぶ.

バージョン

SDL 3.2.0以降
