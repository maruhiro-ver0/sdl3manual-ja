SDL_CreateGPUTransferBuffer

グラフィックリソースのアップロード先またはダウンロード元となる転送バッファを生成する

ヘッダ

SDL3/SDL_gpu.h

構文

SDL_GPUTransferBuffer * SDL_CreateGPUTransferBuffer(SDL_GPUDevice *device, const SDL_GPUTransferBufferCreateInfo *createinfo);

引数

SDL_GPUDevice *deviceGPUコンテキスト
const SDL_GPUTransferBufferCreateInfo *createinfo生成する転送バッファの状態を表す構造体

戻り値

(SDL_GPUTransferBuffer *) 成功のとき転送バッファを戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

ダウンロードバッファの生成は特に高価になりうるため, 定期的にデータをダウンロードする場合は再利用するのがよい.

その他の任意のプロパティはpropsで設定できる. 対応するプロパティ:

  • SDL_PROP_GPU_TRANSFERBUFFER_CREATE_NAME_STRING: デバッグツールに表示される名前

バージョン

SDL 3.2.0以降
