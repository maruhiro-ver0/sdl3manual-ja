SDL_sscanf

sscanf()と同機能だがCランタイムを必要としない関数

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_sscanf(const char *text, const char *fmt, ...);

引数

const char *textスキャンする文字列. NULL不可
const char *fmtprintf形式の文字列. NULL不可
......スキャンした項目の代入先へのポインタの一覧

戻り値

(int) 形式文字列と一致した項目の数を戻す.

詳細

文字列をスキャンし, 形式文字列と一致させ, それぞれの'%'の項目を引数一覧のポインタに代入する.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
