SDL_EnumeratePropertiesCallback

プロパティグループのプロパティの列挙で使用されるコールバックの型

ヘッダ

SDL3/SDL_properties.h

構文

typedef void (SDLCALL *SDL_EnumeratePropertiesCallback)(void *userdata, SDL_PropertiesID props, const char *name);

引数

void *userdataアプリケーションがコールバックに渡したポインタ
SDL_PropertiesIDprops列挙されたプロパティグループ
const char *nameプロパティ名

戻り値

(SDL_PropertyType) プロパティの型を戻す. 設定されていないときSDL_PROPERTY_TYPE_INVALIDを戻す.

詳細

このコールバックはSDL_EnumerateProperties()から1つのプロパティに対して1回ずつ呼ばれる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
