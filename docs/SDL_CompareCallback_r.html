SDL_CompareCallback_r

SDLがソートとバイナリサーチで使用するコールバックの型

ヘッダ

SDL3/SDL_stdinc.h

構文

typedef int (SDLCALL *SDL_CompareCallback_r)(void *userdata, const void *a, const void *b);

引数

void *userdataソート関数に渡したuserdataポインタ
const void *a比較の第1要素へのポインタ
const void *b比較の第2要素へのポインタ

戻り値

(int) aがbの前にソートすべきとき-1, bがaの前にソートすべきとき1, 同等のとき0を戻す. 2つの要素が同等のとき, ソート順は未定義である.
