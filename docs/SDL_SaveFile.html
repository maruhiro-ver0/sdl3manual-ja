SDL_SaveFile

ファイルにデータを書き込む

ヘッダ

SDL3/SDL_iostream.h

構文

bool SDL_SaveFile(const char *file, const void *data, size_t datasize);

引数

const char *fileデータの書き込み先のファイル
const char *data書き込むデータ. datasizeが0のときはNULLでも無効なポインタでもよい
size_tdatasize書き込むバイト数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
