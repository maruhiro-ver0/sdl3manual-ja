SDL_memcpy

メモリをコピーする. 領域が重なっていてはならない

ヘッダ

SDL3/SDL_stdinc.h

構文

void * SDL_memcpy(void *dst, const void *src, size_t len);

引数

const void *dstコピー先の領域. NULLであってはならず, srcと重なっていてはならない
const void *srcコピー元の領域. NULLであってはならず, dstと重なっていてはならない
size_tlendstとsrcのバイト長

戻り値

(void *) dstを戻す.

詳細

メモリ領域は重なっていてはならない. 重なっている場合は, この関数の代わりにSDL_memmove()を使用すること.

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
