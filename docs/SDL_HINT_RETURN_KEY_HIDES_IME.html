SDL_HINT_RETURN_KEY_HIDES_IME

AndroidとiOSでソフトウェアキーボード上でリターンキーを押したときソフトウェアキーボードを非表示にするか否かを設定するヒント

ヘッダ

SDL3/SDL_hints.h

構文

#define SDL_HINT_RETURN_KEY_HIDES_IME "SDL_RETURN_KEY_HIDES_IME"

詳細

このヒントのデフォルトはSDL_PROP_TEXTINPUT_MULTILINE_BOOLEANの値である.

次の値を設定できる:

"0"リターンキーはキーイベントとして扱われる (デフォルト)
"1"リターンキーでキーボードを非表示にする

このヒントはいつでも変更できる.

バージョン

SDL 3.2.0以降
