SDL_UpdateNVTexture

Planar NV12またはNV21テクスチャの領域を新しいピクセルデータで更新する

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_UpdateNVTexture(SDL_Texture *texture, const SDL_Rect *rect, const Uint8 *Yplane, int Ypitch, const Uint8 *UVplane, int UVpitch);

引数

SDL_Texture *texture更新するテクスチャ
const SDL_Rect *rectテクスチャを更新する領域のSDL_Rect. NULLのときテクスチャ全体
const Uint8 *YplaneYプレーンの生のピクセルデータ
intYpitchYプレーンのピクセルデータの幅のバイト数
const Uint8 *UVplaneUVプレーンの生のピクセルデータ
intUVpitchUVプレーンのピクセルデータの幅のバイト数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

NV12/21プレーンのブロックが正しい順序で連続したピクセルデータならばSDL_UpdateTexture()を使用することができるが, この関数はピクセルデータが連続していなくてもよい.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
