SDL_trunc

xの整数部を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_trunc(double x);

引数

doublex浮動小数点の値

戻り値

(double) xの整数部を戻す.

詳細

xを最も近い整数に丸める. これはxの小数部を取り除き整数部のみにすることと同じである.

定義域: -INF <= x <= INF

値域: -INF <= y <= INF, yは整数

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_truncfを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
