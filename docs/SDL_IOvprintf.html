SDL_IOvprintf

データストリームに文字列を書き込む

ヘッダ

SDL3/SDL_iostream.h

構文

size_t SDL_IOvprintf(SDL_IOStream *context, const char *fmt, va_list ap);

引数

SDL_IOStream *contextSDL_IOStreamへのポインタ
const char *fmtprintf()スタイルの文字列形式
va_listap可変長引数

戻り値

(size_t) 書き込んだバイト数を戻す. 失敗したときsizeより小さい場合がある. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数は形式に従って文字列をデータストリームに書き込む.

スレッドセーフ

複数のスレッドから一度に同一のSDL_IOStreamに対して使用してはならない.

バージョン

SDL 3.2.0以降
