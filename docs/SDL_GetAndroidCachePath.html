SDL_GetAndroidCachePath

Androidアプリケーションのキャッシュデータのパス名を得る

ヘッダ

SDL3/SDL_system.h

構文

const char * SDL_GetAndroidCachePath(void);

戻り値

(const char *) 成功のときアプリケーションが使うキャッシュのパス名, 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

パスはアプリケーション固有だが, パブリックで, 他のアプリケーションが書き込むことができる.

典型的なキャッシュのパス名は次のような形式になっている: /data/data/your.app.package/cache/

これはandroid.content.Context.getCacheDir()のCによるラッパーである.

https://developer.android.com/reference/android/content/Context#getCacheDir()

バージョン

SDL 3.2.0以降
