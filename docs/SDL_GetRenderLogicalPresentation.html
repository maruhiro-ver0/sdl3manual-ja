SDL_GetRenderLogicalPresentation

レンダリングのデバイスに依存しないサイズと表示モードを得る

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_GetRenderLogicalPresentation(SDL_Renderer *renderer, int *w, int *h, SDL_RendererLogicalPresentation *mode);

引数

SDL_Renderer *rendererレンダリングコンテキスト
int *w論理幅の代入先へのポインタ
int *h論理高さの代入先へのポインタ
SDL_RendererLogicalPresentation *mode使用中の表示モードの代入先へのポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数は論理レンダリング出力の幅と高さを戻す. 論理サイズが無効のとき0が得られる.

レンダーターゲットにはそれぞれの論理サイズと表示モードがある. この関数は現在のレンダーターゲットの状態を戻す.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
