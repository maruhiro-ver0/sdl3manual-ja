SDL_HINT_OPENVR_LIBRARY

openvr_apiの場所を指定するヒント

ヘッダ

SDL3/SDL_hints.h

構文

#define SDL_HINT_OPENVR_LIBRARY "SDL_OPENGL_LIBRARY"

詳細

デフォルトは, OpenVRドライバを使用する場合はカレントディレクトリでAPIライブラリを検索する. しかし, このヒントで使用するシステムAPIを指定できる. Windowsならば.dll, Linuxならば.soのフルパスまたは相対パスである必要がある.
