SDL_HINT_APP_NAME

アプリケーションの名前を指定するヒント

ヘッダ

SDL3/SDL_hints.h

構文

#define SDL_HINT_APP_NAME "SDL_APP_NAME"

詳細

OSから要求されたとき送るアプリケーションの名称を設定する. これは例えば音声ストリームのボリュームを設定するアプレット内や, スクリーンセーバーの一覧として利用される場合がある. あなたのプログラムの内容を説明する記述("My Game 2: The Revenge"のような)にする必要がある.

アプリケーションで設定した場合, これはSDL_PROP_APP_METADATA_NAME_STRINGで上書きされる.

このヒントはSDLの初期化の前に設定する必要がある.

バージョン

SDL 3.2.0以降
