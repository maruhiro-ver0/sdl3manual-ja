SDL_SetRelativeMouseTransform

相対マウスの入力を変換するユーザ定義の関数を設定する

ヘッダ

SDL3/SDL_mouse.h

構文

bool SDL_SetRelativeMouseTransform(SDL_MouseMotionTransformCallback callback, void *userdata);

引数

SDL_MouseMotionTransformCallbackcallback相対マウスの動きを変換するコールバック. NULLのときデフォルトの動作
void *userdatacallbackに渡すポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

これはシステムの倍率と速度のヒントを上書きする. 先に相対マウスモードを有効にする必要がある. そうでない場合は失敗する.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.4.0以降
