SDL_FillSurfaceRects

指定の色で塗りつぶした複数の長方形を高速に描画する

ヘッダ

SDL3/SDL_surface.h

構文

bool SDL_FillSurfaceRects(SDL_Surface *dst, const SDL_Rect *rects, int count, Uint32 color);

引数

SDL_Surface *dst描画対象のSDL_Surface
const SDL_Rect *rects塗りつぶした長方形を描画する領域を表すSDL_Rectの配列
intcount長方形の配列の数
Uint32color塗りつぶす色

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

colorはサーフェイスのピクセル形式である必要があり, SDL_MapRGB()またはSDL_MapRGBA()で生成できる. 色にα要素が含まれる場合は, ブレンドされるのではなく, 単にα値が描画される.

クリップ枠(SDL_SetSurfaceClipRect()で設定できる)がある場合, クリップ枠とrectの交点に基づいて描画される.

スレッドセーフ

この関数は別のスレッドから別のサーフェイスに対して呼ぶことができる.

バージョン

SDL 3.2.0以降
