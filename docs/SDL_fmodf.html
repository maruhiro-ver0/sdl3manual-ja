SDL_fmodf

浮動小数点のx / yの余りを求める

ヘッダ

SDL3/SDL_stdinc.h

構文

float SDL_fmodf(float x, float y);

引数

floatx分子
floaty分母. 0以外の値であること

戻り値

(float) x / yの余りを戻す.

詳細

xをyで割り, その余りを戻す.

定義域: -INF <= x <= INF, -INF <= y <= INF, y != 0

値域: -INF <= z <= INF

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_fmodを使用すること.

この関数はどのスレッドからも安全に呼べる.
