SDL_SetWindowKeyboardGrab

ウィンドウのマウスの移動範囲を設定する

ヘッダ

SDL3/SDL_video.h

構文

bool SDL_SetWindowMouseRect(SDL_Window *window, const SDL_Rect *rect);

引数

SDL_Window *window移動範囲を設定するウィンドウ
const SDL_Rect *rectウィンドウからの相対座標の長方形領域. NULLのとき, そのウィンドウの移動範囲を破棄する

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

これはカーソルをグラブしないので注意すること. ウィンドウにマウスフォーカスがある場合にカーソルの移動範囲を制限するだけである.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
