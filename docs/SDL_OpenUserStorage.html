SDL_OpenUserStorage

各ユーザのためにファイルシステムを読書可能ストレージコンテナとしてオープンする

ヘッダ

SDL3/SDL_storage.h

構文

SDL_Storage * SDL_OpenUserStorage(const char *org, const char *app, SDL_PropertiesID props);

引数

const char *org組織名
const char *appアプリケーション名
SDL_PropertiesIDpropsバックエンド固有の情報を含むプロパティグループ

戻り値

(SDL_Storage *) 成功のときユーザストレージコンテナを戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

タイトルストレージは一般的に実行中はオープン状態を維持できるが, ユーザストレージはクライアントがファイルの読み書きが可能な場合のみオープンしなければならない. この関数は, ストレージコンテナがクローズしたとき, 背後で適切にファイルのバッチ処理と掃き出しを行うため, 安全で適切なセーブが保証される.

バージョン

SDL 3.2.0以降
