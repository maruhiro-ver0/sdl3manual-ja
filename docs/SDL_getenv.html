SDL_getenv

環境変数の値を得る

ヘッダ

SDL3/SDL_stdinc.h

構文

const char * SDL_getenv(const char *name);

引数

const char *name値を得る環境変数名

戻り値

(const char *) 変数の値へのポインタを戻す. 見つからないときNULLを戻す.

詳細

環境変数名は全ての環境で大文字小文字を区別する.

この関数はSDLがキャッシュした環境のコピーを使用しており, スレッドセーフである.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
