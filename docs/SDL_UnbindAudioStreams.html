SDL_UnbindAudioStreams

一覧のオーディオストリームのオーディオデバイスへのバインドを解除する

ヘッダ

SDL3/SDL_audio.h

構文

void SDL_UnbindAudioStreams(SDL_AudioStream * const *streams, int num_streams);

引数

SDL_AudioStream * const *streamsデバイスへのバインドを解除するオーディオストリームの配列
intnum_streamsstreams配列の要素数

詳細

ストリームは同一のデバイスにバインドされたものである必要はない. 同一のデバイスにバインドされたストリームは自動的に解除される(バインド解除された同一デバイスのストリームは同時にデータの流れが停止する).

バインドされていないデバイスのバインド解除を行おうとすると, 正しく何も起こらない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
