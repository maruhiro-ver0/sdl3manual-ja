SDL_fabsf

xの絶対値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

float SDL_fabsf(float x);

引数

floatx浮動小数点の値

戻り値

(float) xの絶対値を戻す.

詳細

定義域: -INF <= x <= INF

値域: 0 <= y <= INF

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_fabsを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
