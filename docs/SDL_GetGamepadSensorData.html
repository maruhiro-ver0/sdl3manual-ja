SDL_GetGamepadSensorData

ゲームパッドのセンサー状態を得る

ヘッダ

SDL3/SDL_gamepad.h

構文

bool SDL_GetGamepadSensorData(SDL_Gamepad *gamepad, SDL_SensorType type, float *data, int num_values);

引数

SDL_Gamepad *gamepad問い合わせるゲームパッド
SDL_SensorTypetypeセンサーの種類
float *data読み取ったセンサーのデータの代入先へのポインタ
intnum_values書き込まれたデータの数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

値の数とデータの内容はセンサーに依存する. 各センサーの詳細はSDL_SensorTypeの詳細を参照すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
