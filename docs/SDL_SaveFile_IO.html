SDL_SaveFile_IO

データストリームにデータを書き込む

ヘッダ

SDL3/SDL_iostream.h

構文

bool SDL_SaveFile_IO(SDL_IOStream *src, const void *data, size_t datasize, bool closeio);

引数

SDL_IOStream *srcデータを書き込むSDL_IOStream
const char *data書き込むデータ. datasizeが0のときはNULLでも無効なポインタでもよい
size_tdatasize書き込むバイト数
boolcloseio真のとき戻る前にSDL_CloseIO()を呼ぶ(エラーの場合でも)

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

複数のスレッドから一度に同一のSDL_IOStreamに対して使用してはならない.

バージョン

SDL 3.2.0以降
