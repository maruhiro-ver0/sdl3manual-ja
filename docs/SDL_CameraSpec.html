SDL_CameraSpec

カメラデバイスの出力形式の詳細の構造体

ヘッダ

SDL3/SDL_camera.h

フィールド

SDL_PixelFormatformatフレームの形式
SDL_Colorspacecolorspaceフレームの色空間
intwidthフレームの幅
intheightフレームの高さ
intframerate_numeratorフレームレートの分子. (分子/分母 == FPS, 分母/分子 == フレーム間の秒数)
intframerate_denominatorフレームレートの分母. (分子/分母 == FPS, 分母/分子 == フレーム間の秒数)

詳細

カメラは複数の形式に対応している場合が多い. それぞれの形式がこの構造体にカプセル化される.

バージョン

SDL 3.2.0以降
