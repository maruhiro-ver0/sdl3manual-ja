SDL_SavePNG

サーフェイスをPNGファイルとして保存する

ヘッダ

SDL3/SDL_surface.h

構文

bool SDL_SavePNG(SDL_Surface *surface, const char *file);

引数

SDL_Surface *surface画像ファイルとして保存するSDL_Surface
const char *file保存するファイル

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数は別のスレッドから別のサーフェイスに対して呼ぶことができる.

バージョン

SDL 3.4.0以降
