SDL_RenderPoints

レンダーターゲットにサブピクセルの精度の複数の点を描く

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_RenderPoints(SDL_Renderer *renderer, const SDL_FPoint *points, int count);

引数

SDL_Renderer *rendererレンダリングコンテキスト
const SDL_FPoint *points点
intcount点の数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
