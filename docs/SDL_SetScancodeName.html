SDL_SetScancodeName

スキャンコードの人が読める名前を設定する

ヘッダ

SDL3/SDL_keyboard.h

構文

bool SDL_SetScancodeName(SDL_Scancode scancode, const char *name);

引数

SDL_Scancodescancode設定するスキャンコード
const char *nameこのスキャンコードのUTF-8文字列の名前. 文字列はコピーされないため, この関数に渡したポインタはSDLが使用する間は有効でなければならない

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はスレッドセーフではない.

バージョン

SDL 3.2.0以降
