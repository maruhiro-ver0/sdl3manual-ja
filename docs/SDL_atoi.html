SDL_atoi

文字列を解析してint型の数値に変換する

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_atoi(const char *str);

引数

const char *strNUL終端の文字列. NULL不可

戻り値

(double) 解析されたint型の数値を戻す.

詳細

SDL_atoi(str)の結果は(int)SDL_strtol(str, NULL, 10)の結果と同じである.
