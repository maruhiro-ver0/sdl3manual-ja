SDL_SetGamepadPlayerIndex

オープンされたゲームパッドにプレイヤーインデックス番号を設定する

ヘッダ

SDL3/SDL_gamepad.h

構文

bool SDL_SetGamepadPlayerIndex(SDL_Gamepad *gamepad, int player_index);

引数

SDL_Gamepad *gamepadSDL_OpenGamepad()でオープンしたゲームパッドID
intplayer_indexゲームパッドに割り付けるプレイヤーインデックス番号. -1のときプレイヤーインデックス番号を解除してプレイヤーLEDを消灯する

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
