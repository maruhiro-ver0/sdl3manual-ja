SDL_ceil

xを切り上げた値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_ceil(double x);

引数

doublex浮動小数点の値

戻り値

(double) xの切り捨てを戻す.

詳細

xを切り上げた値とは, y >= xを満たす最も小さな整数yのことである.

定義域: -INF <= x <= INF

値域: -INF <= y <= INF, yは整数

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_ceilfを使用すること.

この関数は, バージョン, 環境, 設定で精度が異なる可能性がある. つまり, 同じ入力でも異なるマシンやOSやSDLのバージョン更新で結果が異なることがありうる.

この関数はどのスレッドからも安全に呼べる.
