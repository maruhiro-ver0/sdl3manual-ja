SDL_strrchr

文字列から最後に一致するバイトを検索する

ヘッダ

SDL3/SDL_stdinc.h

構文

char * SDL_strrchr(const char *str, int c);

引数

const char *haystack検索される文字列. NULL不可
intc検索するバイト

戻り値

(char *) needle文字列と最後に一致する箇所へのポインタを戻す. 見つからなかったときNULLを戻す.

詳細

検索は文字列の終端のNUL終端バイトを見つけるまで続く.

この関数が検索するのはバイトであり文字ではないため, 文字コードに関わらずUnicodeコードポイントが255を超える場合は一致しないので注意すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
