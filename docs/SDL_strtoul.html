SDL_strtoul

文字列を解析してunsigned long型の数値に変換する

ヘッダ

SDL3/SDL_stdinc.h

構文

unsigned long SDL_strtoul(const char *str, char **endp, int base);

引数

const char *strNUL終端の文字列. NULL不可
char **endpNULLでない場合, 先頭の無効な文字列(その後の文字列から数値の解析が行われる)の書込先へのポインタ
intbase基数. 0と2から36に対応している. 0の場合, 基数は数字の先頭(0xのとき16進数, 0のとき8進数, その他のとき10進数)による

戻り値

(unsigned long) 解析されたunsigned long型の数値を戻す. 解析できなかった場合0を戻す.

詳細

strが空白で始まる場合, その空白を無視して数値を解析する.

解析された数値がunsigned longの範囲外の場合, 結果はunsigned longで表現可能な最小値または最大値で制限される.
