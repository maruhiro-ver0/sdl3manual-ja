SDL_SetGPUBufferName

バッファのラベルとして任意の文字列定数を設定する

ヘッダ

SDL3/SDL_gpu.h

構文

void SDL_SetGPUBufferName(SDL_GPUDevice *device, SDL_GPUBuffer *buffer, const char *text);

引数

SDL_GPUDevice *deviceGPUコンテキスト
SDL_GPUBuffer *buffer名前を付けるバッファ
const char *textバッファの名前とするUTF-8文字列定数

詳細

この関数のスレッドセーフの問題を避けるには, SDL_CreateGPUBufferでSDL_PROP_GPU_BUFFER_CREATE_NAME_STRINGを使用すること.

スレッドセーフ

この関数はスレッドセーフではない. バッファが他のスレッドで同時に使用されていないことを確認すること.

バージョン

SDL 3.2.0以降
