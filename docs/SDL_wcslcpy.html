SDL_wcslcpy

ワイド文字列をコピーする

ヘッダ

SDL3/SDL_stdinc.h

構文

size_t SDL_wcslcpy(wchar_t *dst, const wchar_t *src, size_t maxlen);

引数

wchar_t *dstコピー先のバッファ. NULLであってはならず, srcと重なっていてはならない
const wchar_t *srcコピーするNUL終端のワイド文字列. NULLであってはならず, dstと重なっていてはならない
size_tmaxlenコピー先のバッファの最大長(ワイド文字数)

戻り値

(size_t) srcの長さ(NUL終端を含まないワイド文字数)を戻す.

詳細

この関数は最大maxlen - 1のワイド文字をsrcからdstにコピーしてNUL終端を追加する.

srcとdstは重なっていてはならない.

maxlenが0の場合, コピーは行われず, NUL終端も書き込まれない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
