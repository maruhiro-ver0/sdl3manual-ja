SDL_GetCurrentRenderOutputSize

現在のレンダリングコンテキストの出力サイズをピクセル数で得る

ヘッダ

SDL3/SDL_render.h

構文

void SDL_GetCurrentRenderOutputSize(SDL_Renderer *renderer);

引数

SDL_Renderer *rendererレンダリングコンテキスト
int *w現在の幅の代入先へのポインタ
int *h現在の高さの代入先へのポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

レンダリングターゲットがアクティブな場合, この関数はレンダリングターゲットのサイズをピクセル数で戻す. アクティブでない場合, SDL_GetRenderOutputSize()の値を戻す.

レンダリングターゲットであってもなくても, 出力はSDL_SetRenderLogicalPresentation()で設定した現在の表示モードで調整される.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
