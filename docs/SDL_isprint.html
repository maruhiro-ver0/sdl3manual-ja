SDL_isprint

文字が表示可能な文字であるかチェックする

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_isprint(int x);

引数

intxチェックする文字

戻り値

(int) xがその分類であれば0以外, そうでなければ0を戻す.

詳細

「表示可能」の定義は黎明期のコンピュータのテキスト端末のもので, Unicode(またはその他の多くの)テキストの扱いとは合わない特殊な場合の関数と考えるべきである.

注意: この関数はシステムロケールとは無関係に英語のASCII文字のみを扱い, ' '(0x20)から'~'(0x7e)のとき真となる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
