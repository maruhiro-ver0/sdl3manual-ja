SDL_sinf

xのサインを求める

ヘッダ

SDL3/SDL_stdinc.h

構文

float SDL_sinf(float x);

引数

floatx浮動小数点のラジアンの値

戻り値

(float) xのサインを戻す.

詳細

定義域: -INF <= x <= INF

値域: -1 <= y <= 1

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_sinを使用すること.

この関数は, バージョン, 環境, 設定で精度が異なる可能性がある. つまり, 同じ入力でも異なるマシンやOSやSDLのバージョン更新で結果が異なることがありうる.
