SDL_CreateGPURenderState

GPUレンダリング状態を生成する

ヘッダ

SDL3/SDL_render.h

構文

SDL_GPURenderState * SDL_CreateGPURenderState(SDL_Renderer *renderer, const SDL_GPURenderStateCreateInfo *createinfo);

引数

SDL_GPURenderState *renderer使用するレンダラ
const SDL_GPURenderStateCreateInfo *createinfo生成するGPUレンダラ状態のパラメータ

戻り値

(SDL_GPURenderState *) GPUレンダリング状態を戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はレンダラを生成したスレッドと同じスレッドで呼ぶ必要がある.

バージョン

SDL 3.4.0以降
