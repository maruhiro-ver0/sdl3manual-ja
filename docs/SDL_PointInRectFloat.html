SDL_PointInRectFloat

浮動小数点で表された点が長方形の内側にあるかチェックする

ヘッダ

SDL3/SDL_rect.h

構文

SDL_FORCE_INLINE bool SDL_PointInRectFloat(const SDL_FPoint *p, const SDL_FRect *r);

引数

const SDL_Point *pチェックする点
const SDL_Rect *rチェックする長方形

戻り値

(bool) pがrの内側にあるとき真, そうでないとき偽を戻す.

詳細

pとrのどちらもNULLでなく, かつpのXとYが長方形の左上以上で, かつ長方形のX+WとY+H以下のとき, 点は長方形の内側にあるとみなされる. そのため, 1×1の長方形の(0,0)と(0,1)は「内側」だが, (0,2)はそうではない.

この関数はヘッダにある強制インライン関数で, SDLライブラリのパブリックAPI関数として有効ではないので注意すること. (言い換えると, このコードは呼び出したプログラムに組み込まれるので, リンカや動的ローダはSDL内部にこの関数を見つけることができない.)

バージョン
