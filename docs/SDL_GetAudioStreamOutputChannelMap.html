SDL_GetAudioStreamOutputChannelMap

オーディオストリームの出力チャネルマップを得る

ヘッダ

SDL3/SDL_audio.h

構文

int * SDL_GetAudioStreamOutputChannelMap(SDL_AudioStream *stream, int *count);

引数

SDL_AudioStream *stream問い合わせるオーディオストリーム
const int *chmap新しいチャネルマップ. NULLのときデフォルトに戻す
int *countチャネルの数の代入先へのポインタ. NULLも可

戻り値

(int *) 現在の出力チャネルと同じ要素数の現在のチャネルマップの配列を戻す. デフォルトのときNULLを戻す. 不要になったときSDL_free()で解放する必要がある.

詳細

チャネルマップは任意で, ほとんどの場合は必要なく, SDLが求める順序でデータを渡せばよい.

オーディオストリームはデフォルトでは再マップが適用されていない. NULLはこの状態を表すもので, エラーではない

スレッドセーフ

この関数はストリーム固有のミューテックスが行われている限りどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
