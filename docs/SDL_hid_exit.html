SDL_hid_exit

HIDAPIライブラリを終了する

ヘッダ

SDL3/SDL_hidapi.h

構文

int SDL_hid_exit(void);

戻り値

(int) 成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

詳細

この関数はHIDAPIに関連する全ての静的データを解放する. メモリリークを避けるため, 実行を終了するとき呼ぶ必要がある.

バージョン

SDL 3.2.0以降
