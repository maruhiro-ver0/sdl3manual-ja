SDL_floorf

xを切り捨てた値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

float SDL_floorf(float x);

引数

floatx浮動小数点の値

戻り値

(float) xを切り捨てた値を戻す.

詳細

xを切り捨てた値とは, y <= xを満たす最も大きな整数yのことである.

定義域: -INF <= x <= INF

値域: -INF <= y <= INF, yは整数

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_floorを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン
