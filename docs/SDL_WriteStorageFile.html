SDL_WriteStorageFile

クライアントのメモリからストレージコンテナのファイルに同期的に読み込む

ヘッダ

SDL3/SDL_storage.h

構文

bool SDL_WriteStorageFile(SDL_Storage *storage, const char *path, const void *source, Uint64 length);

引数

SDL_Storage *storage書き込むストレージコンテナ
const char *path書き込むファイルの相対パス
void *destinationファイルに書き込むクライアントのバッファ
Uint64lengthバッファの長さ

戻り値

(bool) ファイルの書き込みに成功したとき真, 失敗したとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
