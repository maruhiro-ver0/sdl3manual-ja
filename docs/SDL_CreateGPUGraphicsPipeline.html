SDL_CreateGPUGraphicsPipeline

グラフィックワークフローで使用されるパイプラインオブジェクトを生成する

ヘッダ

SDL3/SDL_gpu.h

構文

SDL_GPUGraphicsPipeline * SDL_CreateGPUGraphicsPipeline(SDL_GPUDevice *device, const SDL_GPUGraphicsPipelineCreateInfo *createinfo);

引数

SDL_GPUDevice *deviceGPUコンテキスト
const SDL_GPUGraphicsPipelineCreateInfo *createinfo生成するグラフィックパイプラインの状態を表す構造体

戻り値

(SDL_GPUGraphicsPipeline *) 成功のときグラフィックパイプラインオブジェクトを戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

その他の任意のプロパティはpropsで設定できる. 対応するプロパティ:

  • SDL_PROP_GPU_GRAPHICSPIPELINE_CREATE_NAME_STRING: デバッグツールに表示される名前

バージョン

SDL 3.2.0以降
