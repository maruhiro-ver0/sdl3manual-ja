SDL_scalbn

2の整数乗のx倍を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_scalbn(double x, int n);

引数

doublex浮動小数点の倍数
intn整数の指数

戻り値

(double) x * 2^nを戻す.

詳細

浮動小数点の基数(常に2)のn乗にxを乗算した値.

定義域: -INF <= x <= INF, nは整数

値域: MIN_LONG <= y <= MAX_LONG

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_scalbnfを使用すること.
