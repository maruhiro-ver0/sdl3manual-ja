SDL_sin

xのサインを求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_sin(double x);

引数

doublex浮動小数点のラジアンの値

戻り値

(double) xのサインを戻す.

詳細

定義域: -INF <= x <= INF

値域: -1 <= y <= 1

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_sinfを使用すること.

この関数は, バージョン, 環境, 設定で精度が異なる可能性がある. つまり, 同じ入力でも異なるマシンやOSやSDLのバージョン更新で結果が異なることがありうる.
