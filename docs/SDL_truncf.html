SDL_truncf

xの整数部を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

float SDL_truncf(float x);

引数

floatx浮動小数点の値

戻り値

(float) xの整数部を戻す.

詳細

xを最も近い整数に丸める. これはxの小数部を取り除き整数部のみにすることと同じである.

定義域: -INF <= x <= INF

値域: -INF <= y <= INF, yは整数

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_truncを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
