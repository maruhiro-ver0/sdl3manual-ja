SDL_SetGamepadMapping

ジョイスティックまたはゲームパッドのマッピングを設定する

ヘッダ

SDL3/SDL_gamepad.h

構文

bool SDL_SetGamepadMapping(SDL_JoystickID instance_id, const char *mapping);

引数

SDL_JoystickIDinstance_idジョイスティックインスタンスID
const char *mapping使用するマッピング文字列. NULLのときマッピングをクリアする

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

マッピングの詳細はSDL_AddGamepadMapping()を参照すること.

バージョン

SDL 3.2.0以降
