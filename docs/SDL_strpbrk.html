SDL_strpbrk

文字列から最初に一致する文字セット内の文字を検索し, その文字の位置へのポインタを戻す

ヘッダ

SDL3/SDL_stdinc.h

構文

char * SDL_strpbrk(const char *str, const char *breakset);

引数

const char *str検索される文字列. NULLであってはならず, breaksetと重なっていてはならない
const char *breakset検索する文字の一覧を表すNUL終端の文字列. NULLであってはならず, strと重なっていてはならない

戻り値

(char *) str文字列のbreakset内の文字と最初に一致する箇所へのポインタを戻す. 一致しないときNULLを戻す.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
