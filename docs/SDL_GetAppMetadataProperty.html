SDL_GetAppMetadataProperty

アプリケーションのメタデータを得る

ヘッダ

SDL3/SDL_init.h

構文

const char * SDL_GetAppMetadataProperty(const char *name);

引数

const char *nameメタデータプロパティ名

戻り値

(const char *) メタデータプロパティの現在の値を戻す. 設定されていないときデフォルトを戻す. デフォルトがないときNULLを戻す.

詳細

この関数は以前SDL_SetAppMetadata()またはSDL_SetAppMetadataProperty()で設定したメタデータを戻す. 有効なプロパティとその意味の一覧はSDL_SetAppMetadataProperty()にある.

スレッドセーフ

この関数はどのスレッドからも安全に呼べるが, 戻された文字列は保護されておらず, 別スレッドでそのプロパティを設定するためにSDL_SetAppMetadataProperty()が呼ばれて解放される可能性がある.

バージョン

SDL 3.2.0以降
