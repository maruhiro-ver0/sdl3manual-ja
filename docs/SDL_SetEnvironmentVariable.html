SDL_SetEnvironmentVariable

環境変数セットの環境変数の値を設定する

ヘッダ

SDL3/SDL_stdinc.h

構文

bool SDL_SetEnvironmentVariable(SDL_Environment *env, const char *name, const char *value, bool overwrite);

引数

SDL_Environment *env変更する環境変数セット
const char *name値を設定する環境変数名
const char *value環境変数に設定する値
booloverwrite真のとき既に変数が存在するとき上書きする. 偽のとき既に変数が存在するとき値を設定せずに成功を戻す

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.
