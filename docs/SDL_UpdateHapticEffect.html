SDL_UpdateHapticEffect

ハプティックデバイスのエフェクトのパラメータを変更する

ヘッダ

SDL3/SDL_haptic.h

構文

bool SDL_UpdateHapticEffect(SDL_Haptic *haptic, SDL_HapticEffectID effect, const SDL_HapticEffect *data);

引数

SDL_Haptic *haptic変更するハプティックデバイス
SDL_HapticEffectIDeffect変更するエフェクトのID
const SDL_HapticEffect *data新しいエフェクトのパラメータ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

これは動的に更新できるが, 方向が正しくなくなることがある. エフェクトを再アップロードした後, 明示的に再生を開始したほうがよい. SDL_UpdateHapticEffect()で再生している最中は更新できない.

バージョン

SDL 3.2.0以降
