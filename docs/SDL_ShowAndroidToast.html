SDL_ShowAndroidToast

Androidトースト通知を表示する

ヘッダ

SDL3/SDL_system.h

構文

bool SDL_ShowAndroidToast(Uint32 command, int param);

引数

const char *message表示するテキストメッセージ
intduration0=短い, 1=長い
intgravity通知を表示する画面上の位置
intxoffsetgravity>=0の場合のみこの引数が設定される
intyoffsetgravity>=0の場合のみこの引数が設定される

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

トーストはAndroid固有の軽い注意喚起の一種である.

https://developer.android.com/guide/topics/ui/notifiers/toasts

ーストはUIスレッドで表示される.

引数gravityは下の値から選ぶこと. 適当なものがない場合は-1とすること.

https://developer.android.com/reference/android/view/Gravity

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
