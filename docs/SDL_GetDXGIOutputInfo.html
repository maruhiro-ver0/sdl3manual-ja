SDL_GetDXGIOutputInfo

指定のディスプレイのDXGIアダプタと出力のインデックス番号の一覧を得る

ヘッダ

SDL3/SDL_system.h

構文

bool SDL_GetDXGIOutputInfo(SDL_DisplayID displayID, int *adapterIndex, int *outputIndex);

引数

SDL_DisplayIDdisplayID一覧を得るディスプレイのインスタンスID
int *adapterIndexアダプタのインデックス番号の代入へのポインタ
int *outputIndex出力のインデックス番号の代入先へのポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

DXGIアダプタと出力のインデックス番号の一覧は, DX10またはDX11デバイスとスワップチェーンを生成するために必要なオブジェクトを得るために, EnumAdaptersとEnumOutputsに1つずつ渡すことができる.

バージョン

SDL 3.2.0以降
