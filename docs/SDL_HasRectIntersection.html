SDL_HasRectIntersection

2つの長方形に辺の交点があるかチェックする

ヘッダ

SDL3/SDL_rect.h

構文

bool SDL_HasRectIntersection(const SDL_Rect *A, const SDL_Rect *B);

引数

const SDL_Rect *A1つ目の長方形
const SDL_Rect *B2つ目の長方形

戻り値

(bool) 辺が交わるとき真, 交わらないとき偽を戻す.

詳細

どちらかのポインタがNULLの場合, この関数は偽を戻す.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
