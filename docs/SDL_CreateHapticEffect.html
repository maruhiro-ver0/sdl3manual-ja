SDL_CreateHapticEffect

ハプティックデバイスに新しいエフェクトを生成する

ヘッダ

SDL3/SDL_haptic.h

構文

SDL_HapticEffectID SDL_CreateHapticEffect(SDL_Haptic *haptic, const SDL_HapticEffect *effect);

引数

SDL_Haptic *hapticエフェクトを生成するハプティックデバイス
const SDL_HapticEffect *effect生成するエフェクトのSDL_HapticEffect

戻り値

(SDL_HapticEffectID) 成功のときハプティックエフェクトIDを戻す. 失敗のとき-1を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
