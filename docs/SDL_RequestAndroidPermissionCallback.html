SDL_RequestAndroidPermissionCallback

SDL_RequestAndroidPermissionの応答で呼ばれるコールバックの型

ヘッダ

SDL3/SDL_system.h

構文

typedef void (SDLCALL *SDL_RequestAndroidPermissionCallback)(void *userdata, const char *permission, bool granted);

引数

void *userdataアプリケーションがコールバックに渡したポインタ
const char *permission要求したAndroid固有の権限名
boolgranted許可されたとき真. 拒否されたとき偽
