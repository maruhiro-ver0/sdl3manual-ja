SDL_GetRenderOutputSize

レンダリングコンテキストの出力サイズをピクセル数で得る

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_GetRenderOutputSize(SDL_Renderer *renderer, int *w, int *h);

引数

SDL_Renderer *rendererレンダリングコンテキスト
int *w幅ピクセル数の代入先へのポインタ
int *h高さピクセル数の代入先へのポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数は実際の出力サイズをピクセル数で戻す. レンダーターゲットや論理サイズと表示モードは無視される.

現在のレンダリングターゲットの出力サイズは論理サイズに調整される. この関数の代わりにSDL_GetCurrentRenderOutputSize()を使用すること.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
