SDL_SetPrimarySelectionText

プライマリ選択にUTF-8テキストを設定する

ヘッダ

SDL3/SDL_clipboard.h

構文

bool SDL_SetPrimarySelectionText(const char *text);

引数

const char *textプライマリ選択に書き込むテキスト

戻り値

(bool) 成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
