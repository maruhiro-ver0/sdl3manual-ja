SDL_GetPropertyType

プロパティグループのプロパティの型を得る

ヘッダ

SDL3/SDL_properties.h

構文

SDL_PropertyType SDL_GetPropertyType(SDL_PropertiesID props, const char *name);

引数

SDL_PropertiesIDprops調査するプロパティグループ
const char *name調査するプロパティの名前

戻り値

(SDL_PropertyType) プロパティの型を戻す. 設定されていないときSDL_PROPERTY_TYPE_INVALIDを戻す.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
