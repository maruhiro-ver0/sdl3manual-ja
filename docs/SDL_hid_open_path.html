SDL_hid_open_path

HIDデバイスを名前で開く

ヘッダ

SDL3/SDL_hidapi.h

構文

SDL_hid_device * SDL_hid_open_path(const char *path);

引数

const char *pathオープンするデバイスのパス名

戻り値

(SDL_hid_device *)SDL_hid_deviceオブジェクトへのポインタを戻す. 失敗のときNULLを戻す SDL_GetError()で詳細を知ることができる.

詳細

パス名はSDL_hid_enumerate()を呼んで決める. 環境固有のパス名(例: Linuxならば /dev/hidraw0 等)も使える.

バージョン

SDL 3.2.0以降
