SDL_vsscanf

vsscanf()と同機能だがCランタイムを必要としない関数

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_vsscanf(const char *text, const char *fmt, va_list ap);

引数

const char *textスキャンする文字列. NULL不可
const char *fmtprintf形式の文字列. NULL不可
va_listapスキャンした項目の代入先へのポインタのva_list

戻り値

(int) 形式文字列と一致した項目の数を戻す.

詳細

この関数は...ではなくva_listを使うことを以外はSDL_sscanf()と同じである.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
