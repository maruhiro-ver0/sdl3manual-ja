SDL_floor

xを切り捨てた値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

double SDL_floor(double x);

引数

doublex浮動小数点の値

戻り値

(double) xを切り捨てた値を戻す.

詳細

xを切り捨てた値とは, y <= xを満たす最も大きな整数yのことである.

定義域: -INF <= x <= INF

値域: -INF <= y <= INF, yは整数

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_floorfを使用すること.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
