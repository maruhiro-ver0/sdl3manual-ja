SDL_HapticEffectSupported

ハプティックデバイスがエフェクトに対応しているかチェックする

ヘッダ

SDL3/SDL_haptic.h

構文

bool SDL_HapticEffectSupported(SDL_Haptic *haptic, const SDL_HapticEffect *effect);

引数

SDL_Haptic *hapticチェックするハプティックデバイス
const SDL_HapticEffect *effectチェックするエフェクト

戻り値

(bool) エフェクトに対応していれば真, していなければ偽を戻す.

バージョン

SDL 3.2.0以降
