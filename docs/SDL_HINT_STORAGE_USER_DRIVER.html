SDL_HINT_STORAGE_USER_DRIVER

ユーザストレージで使用するバックエンドを設定するヒント

ヘッダ

SDL3/SDL_hints.h

構文

#define SDL_HINT_STORAGE_USER_DRIVER "SDL_STORAGE_USER_DRIVER"

詳細

デフォルトでは, SDLは有効な全てのストレージバックエンドを利用可能なものを見つけるまで適切な順序で試すが, このヒントでアプリケーションまたはユーザが指定したバックエンドを強制できる. 例えば, Steam上だがユーザデータでSteamRemoteStorageを使用したくない場合, "pc"のようにする.

このヒントはSDLの初期化の前に設定する必要がある.

バージョン

SDL 3.2.0以降
