SDL_BlitSurfaceScaled

サーフェイスに拡大縮小して転送する. 形式が異なってもよい

ヘッダ

SDL3/SDL_surface.h

構文

bool SDL_BlitSurfaceScaled(SDL_Surface *src, const SDL_Rect *srcrect, SDL_Surface *dst, const SDL_Rect *dstrect, SDL_ScaleMode scaleMode);

引数

SDL_Surface *srcコピー元のSDL_Surface
const SDL_Rect *srcrectサーフェイスのコピー元の領域のSDL_Rect. NULLのときサーフェイス全体
SDL_Surface *dstコピー先のSDL_Surface
const SDL_Rect *dstrectサーフェイスのコピー先の領域を表すSDL_Rect. NULLのときサーフェイス全体
SDL_ScaleModescaleMode使用する拡大縮小アルゴリズム

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

srcとdstのスレッドを同時に使用するのは1つのスレッドである必要がある

バージョン

SDL 3.2.0以降
