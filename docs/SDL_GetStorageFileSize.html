SDL_GetStorageFileSize

ストレージコンテナのファイルのサイズを得る

ヘッダ

SDL3/SDL_storage.h

構文

bool SDL_GetStorageFileSize(SDL_Storage *storage, const char *path, Uint64 *length);

引数

SDL_Storage *storage問い合わせるストレージコンテナ
const char *pathサイズを求めるファイルの相対パス
Uint64lengthファイルサイズの代入先へのポインタ

戻り値

(bool) ファイルサイズを得られたとき真, 失敗したとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
