SDL_GetTrayEntries

トレイのメニューの項目の一覧を順序通りに得る

ヘッダ

SDL3/SDL_tray.h

構文

const SDL_TrayEntry ** SDL_GetTrayEntries(SDL_TrayMenu *menu, int *count);

引数

SDL_TrayMenu *menu項目の一覧を求めるメニュー
int *menuメニューの項目数の代入先へのポインタ. NULLも可

戻り値

(const SDL_TrayEntry **) メニューの項目の一覧を表すNULL終端の配列を戻す. メニューへの項目の追加や削除を行う関数を呼ぶと, この一覧は無効になる.

スレッドセーフ

この関数はトレイを生成したスレッドで呼ぶ必要がある.

バージョン

SDL 3.2.0以降
