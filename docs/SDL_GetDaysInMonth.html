SDL_GetDaysInMonth

与えられた年の月の日数を求める

ヘッダ

SDL3/SDL_time.h

構文

int SDL_GetDaysInMonth(int year, int month);

引数

intyear年
intmonth月 [1-12]

戻り値

(int) その月の日数を戻す. 月が不正のとき-1を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
