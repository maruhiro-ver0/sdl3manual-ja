SDL_GetGamepadTouchpadFinger

ゲームパッドのタッチパッド上の指の状態を得る

ヘッダ

SDL3/SDL_gamepad.h

構文

bool SDL_GetGamepadTouchpadFinger(SDL_Gamepad *gamepad, int touchpad, int finger, bool *down, float *x, float *y, float *pressure);

引数

SDL_Gamepad *gamepadゲームパッド
inttouchpadタッチパッドのインデックス番号
intfingerタッチパッド上の指のインデックス番号
bool *down指を当てているとき真, 離しているとき偽の代入先へのポインタ. NULLも可
float *xタッチパッド上の指のX座標の代入先へのポインタ. 左上を原点として0から1で正規化した値. NULLも可
float *yタッチパッド上の指のY座標の代入先へのポインタ. 左上を原点として0から1で正規化した値. NULLも可
float *pressure指の圧力の代入先へのポインタ. NULLも可

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
