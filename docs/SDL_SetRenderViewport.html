SDL_SetRenderViewport

現在のレンダーターゲットの描画領域を設定する

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_SetRenderViewport(SDL_Renderer *renderer, const SDL_Rect *rect);

引数

SDL_Renderer *rendererレンダリングコンテキスト
const SDL_Rect *rect描画領域を表す長方形. NULLのとき描画領域を全体とする

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

描画はこの領域でクリップされ(これとは別にSDL_SetRenderClipRectでもクリップされる), 描画領域の左上が以降の描画命令の(0, 0)となる.

描画領域の幅と高さは0以上の必要がある.

レンダーターゲットにはそれぞれの描画領域がある. この関数は現在のレンダーターゲットの描画領域を設定する.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
