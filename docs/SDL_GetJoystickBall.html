SDL_GetJoystickBall

トラックボールの最後のポーリングからの方向の変化量を得る

ヘッダ

SDL3/SDL_joystick.h

構文

bool SDL_GetJoystickBall(SDL_Joystick *joystick, int ball, int *dx, int *dy);

引数

SDL_Joystick *joystickジョイスティックの情報を持つSDL_Joystick
intball問い合わせるトラックボール. インデックス番号は0から始まる
int *dx最後のポーリングからのX軸方向の変化量
int *dy最後のポーリングからのY軸方向の変化量

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

トラックボールからは最後にSDL_GetJoystickBall()を呼んだ時点からの相対値しか得られない. 変化量はdxとdyに代入される.

多くのジョイスティックにはトラックボールは存在しない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
