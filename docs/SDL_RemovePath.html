SDL_RemovePath

ファイルまたは空のディレクトリを削除する

ヘッダ

SDL3/SDL_filesystem.h

構文

bool SDL_RemovePath(const char *path);

引数

const char *pathファイルシステムから削除するパス

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

空でないディレクトリの場合, 失敗する. そのため, この関数はディレクトリツリーを再帰的に削除しない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
