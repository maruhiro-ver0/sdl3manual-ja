SDL_PenInputFlags

様々なペンイベントのpen_stateフィールドで使用されるペン入力フラグの型

ヘッダ

SDL3/SDL_pen.h

構文

typedef Uint32 SDL_PenInputFlags;

値

SDL_PEN_INPUT_DOWN1u << 0ペンを押し付けた
SDL_PEN_INPUT_BUTTON_11u << 1ボタン1が押された
SDL_PEN_INPUT_BUTTON_21u << 2ボタン2が押された
SDL_PEN_INPUT_BUTTON_31u << 3ボタン3が押された
SDL_PEN_INPUT_BUTTON_41u << 4ボタン4が押された
SDL_PEN_INPUT_BUTTON_51u << 5ボタン5が押された
SDL_PEN_INPUT_ERASER_TIP1u << 30消しゴムが使われた
SDL_PEN_INPUT_IN_PROXIMITY1u << 31ペンを近づけた (SDL 3.4.0以降)

バージョン

SDL 3.2.0以降
