SDL_RemoveHintCallback

指定のヒントを監視する関数を取り除く

ヘッダ

SDL3/SDL_hints.h

構文

void SDL_RemoveHintCallback(const char* name, SDL_HintCallback callback, void* userdata)

引数

const char *name監視するヒント
SDL_HintCallbackcallbackヒントの値が変わったとき呼ぶ関数
void *userdataコールバック関数に渡すポインタ

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.
