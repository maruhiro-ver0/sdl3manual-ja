SDL_GetAudioDeviceChannelMap

オーディオデバイスの現在のチャネルマップを得る

ヘッダ

SDL3/SDL_audio.h

構文

int * SDL_GetAudioDeviceChannelMap(SDL_AudioDeviceID devid, int *count);

引数

SDL_AudioDeviceIDdevid調査するオーディオデバイスのインスタンスID
int *countチャネルの数の代入先へのポインタ. NULLも可

戻り値

(int *) 現在の出力チャネルの数と同じ要素数の現在のチャネルマップの配列を戻す. デフォルトの場合NULLを戻す. これは必要がなくなったときSDL_free()で解放する必要がある.

詳細

チャネルマップは任意で, ほとんどの場合は必要なく, SDLが求める順序でデータを渡せばよい.

通常, オーディオデバイスには再マップが適用されていない. この場合はNULLが戻るが, エラーではない.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
