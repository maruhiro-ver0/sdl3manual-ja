SDL_memcmp

2つのメモリバッファを比較する

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_memcmp(const void *s1, const void *s2, size_t len);

引数

const void *s1比較する1つ目のバッファ. NULL不可
const void *s2比較する2つ目のバッファ. NULL不可
size_tlen比較するバッファのバイト長

戻り値

(int) s1がs2「未満」のとき負の数, s1がs2を「超える」とき正の数, バッファの内容がlenで一致するとき0を戻す.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
