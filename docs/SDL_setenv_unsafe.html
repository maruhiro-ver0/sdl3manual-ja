SDL_setenv_unsafe

環境変数に値を設定する

ヘッダ

SDL3/SDL_stdinc.h

構文

int SDL_setenv_unsafe(const char *name, const char *value, int overwrite);

引数

const char *name値を設定する環境変数名
const char *value環境変数に設定する値
intoverwrite1のとき既に変数が存在するとき上書きする. 0のとき既に変数が存在するとき値を設定せずに成功を戻す

戻り値

(int) 成功のとき0, 失敗のとき-1を戻す.

スレッドセーフ

この関数はスレッドセーフではない. 代わりにSDL_SetEnvironmentVariable()の使用を検討すること.

バージョン

SDL 3.2.0以降
