SDL_SetRenderVSync

レンダラの垂直同期の方法を設定する

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_SetRenderVSync(SDL_Renderer *renderer, int vsync);

#define SDL_RENDERER_VSYNC_DISABLED 0
#define SDL_RENDERER_VSYNC_ADAPTIVE (-1)

引数

SDL_Renderer *rendererレンダリングコンテキスト
intvsync垂直同期の周期

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

レンダラが生成されたとき, 垂直同期のデフォルトはSDL_RENDERER_VSYNC_DISABLEDである.

vsyncが1のとき垂直同期信号のたびに表示を行い, 2のとき2回の垂直同期信号で1回表示を行う, のようになっており, SDL_RENDERER_VSYNC_ADAPTIVEのときlate swap tearing(adaptive vsync(適応垂直同期)), SDL_RENDERER_VSYNC_DISABLEDのとき垂直同期を無効にする. 全てのドライバが全ての値に対応しているわけではない. SDL_GetRenderVSync()が戻した値を比較して, 要求した設定に対応しているかチェックする必要がある.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
