SDL_GetGamepadStringForType

SDL_GamepadType列挙体の値から文字列を得る

ヘッダ

SDL3/SDL_gamepad.h

構文

const char * SDL_GetGamepadStringForType(SDL_GamepadType type);

引数

SDL_GamepadTypetypeSDL_GamepadType列挙体の値

戻り値

(const char *) 渡したゲームパッドの種類の文字列を戻す. 無効な方向のときNULLを戻す. 文字列はSDL_Gamepadマッピング文字列の形式である.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
