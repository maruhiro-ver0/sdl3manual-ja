SDL_GetTouchFingers

タッチデバイス上の指の一覧を得る

ヘッダ

SDL3/SDL_pen.h

構文

SDL_Finger ** SDL_GetTouchFingers(SDL_TouchID touchID, int *count);

引数

SDL_TouchIDtouchIDタッチデバイスインスタンスID
int *countデバイスの数の代入先へのポインタ. NULLも可

戻り値

(SDL_TouchID **) NULLが終端の指IDの配列を戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる. 不要になったときSDL_free()で解放する必要がある.

バージョン

SDL 3.2.0以降
