SDL_RenderRects

レンダーターゲットにサブピクセルの精度の複数の長方形を描く

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_RenderRects(SDL_Renderer *renderer, const SDL_FRect *rects, int count);

引数

SDL_Renderer *rendererレンダリングコンテキスト
const SDL_FRect *rects長方形のSDL_FRectの配列
intcount長方形の数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降
