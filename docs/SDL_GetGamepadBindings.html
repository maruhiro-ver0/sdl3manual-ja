SDL_GetGamepadBindings

ゲームパッドにバインドされたSDLジョイスティックレイヤを得る

ヘッダ

SDL3/SDL_gamepad.h

構文

SDL_GamepadBinding ** SDL_GetGamepadBindings(SDL_Gamepad *gamepad, int *count);

引数

SDL_Gamepad *gamepadゲームパッド
int *countバインド情報の数の代入先へのポインタ

戻り値

(SDL_GamepadBinding **) NULL終端のバインド情報ポインタの配列を戻す. 失敗のときNULLを戻す. SDL_GetError()を呼んで詳細を知ることができる. これは1回のメモリ確保を行っているため, 不要になったときSDL_free()で解放する必要がある.

スレッドセーフ

この関数はどのスレッドからも安全に呼べる.

バージョン

SDL 3.2.0以降
