SDL_lroundf

xを四捨五入したlong型の値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

long SDL_lroundf(float x);

引数

floatx浮動小数点の値

戻り値

(long) xを四捨五入した値を戻す.

詳細

xを最も近い整数に丸める. 中間にある値は0に近い整数にする.

定義域: -INF <= x <= INF

値域: MIN_LONG <= y <= MAX_LONG

この関数は単精度浮動小数点の値を扱う. 倍精度の場合はSDL_lroundを使用すること. 結果を浮動小数点としたい場合はSDL_roundfを使用すること.
