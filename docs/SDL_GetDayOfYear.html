SDL_GetDayOfYear

カレンダー日付から通算日数を得る

ヘッダ

SDL3/SDL_time.h

構文

int SDL_GetDayOfYear(int year, int month, int day);

引数

intyear日付の年
intmonth日付の月
intday日付の日

戻り値

(int) 0から365の通算日数を戻す. 日付が不正のとき-1を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
