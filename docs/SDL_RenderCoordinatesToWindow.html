SDL_RenderCoordinatesToWindow

レンダリングの座標をウィンドウの座標に変換する

ヘッダ

SDL3/SDL_render.h

構文

bool SDL_RenderCoordinatesToWindow(SDL_Renderer *renderer, float x, float y, float *window_x, float *window_y);

引数

SDL_Renderer *rendererレンダリングコンテキスト
floatwindow_xレンダラ座標系のX座標
floatwindow_yレンダラ座標系のY座標
float *xウィンドウ座標系に変換したX座標の代入先へのポインタ
float *yウィンドウ座標系に変換したY座標の代入先へのポインタ

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

これはいくつかの状態を考慮する:

  • ウィンドウのサイズ
  • 表示モードの設定 (SDL_SetRenderLogicalPresentation)
  • 倍率 (SDL_SetRenderScale)
  • 描画領域 (SDL_SetRenderViewport)

スレッドセーフ

この関数を呼べるのはメインスレッドのみである.

バージョン

SDL 3.2.0以降

関連項目
