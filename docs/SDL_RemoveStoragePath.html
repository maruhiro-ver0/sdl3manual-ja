SDL_RemoveStoragePath

書込可能ストレージコンテナのファイルまたは空のディレクトリを削除する

ヘッダ

SDL3/SDL_storage.h

構文

bool SDL_RemoveStoragePath(SDL_Storage *storage, const char *path);

引数

SDL_Storage *storageストレージコンテナ
const char *pathファイルシステムから削除するパス

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 3.2.0以降
