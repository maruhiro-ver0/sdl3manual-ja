SDL_SetTrayTooltip

システムトレイのツールチップを設定する

ヘッダ

SDL3/SDL_tray.h

構文

void SDL_SetTrayTooltip(SDL_Tray *tray, const char *tooltip);

引数

SDL_Tray *trayアイコンを設定するシステムトレイ
const char *tooltip新しいツールチップのUTF-8文字列. NULLも可

スレッドセーフ

この関数はトレイを生成したスレッドで呼ぶ必要がある.

バージョン

SDL 3.2.0以降
