SDL_SetTrayEntryLabel

トレイの項目のラベルを設定する

ヘッダ

SDL3/SDL_tray.h

構文

void SDL_SetTrayEntryLabel(SDL_TrayEntry *entry, const char *label);

引数

SDL_TrayEntry *entryラベルを設定するトレイの項目
const char *labelラベルのUTF-8文字列

詳細

トレイの項目は, それぞれ区切りと通常の項目に変えることはできない. つまり, ラベルがNULL(区切り)の項目にNULL以外のラベルを設定することや, ラベルがNULLの項目にNULL以外のラベルを設定することはできない. それを行うと, この関数は何もせず失敗する.

スレッドセーフ

この関数はトレイを生成したスレッドで呼ぶ必要がある.

バージョン

SDL 3.2.0以降
