SDL_lround

xを四捨五入したlong型の値を求める

ヘッダ

SDL3/SDL_stdinc.h

構文

long SDL_lround(double x);

引数

doublex浮動小数点の値

戻り値

(long) xを四捨五入した値を戻す.

詳細

xを最も近い整数に丸める. 中間にある値は0に近い整数にする.

定義域: -INF <= x <= INF

値域: MIN_LONG <= y <= MAX_LONG

この関数は倍精度浮動小数点の値を扱う. 単精度の場合はSDL_lroundfを使用すること. 結果を浮動小数点としたい場合はSDL_roundを使用すること.

この関数はどのスレッドからも安全に呼べる.

バージョン
