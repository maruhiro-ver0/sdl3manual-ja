SDL_GetSensorData

オープンしたセンサーの現在の状態を得る

ヘッダ

SDL3/SDL_sensor.h

構文

bool SDL_GetSensorData(SDL_Sensor *sensor, float *data, int num_values);

引数

SDL_Sensor *sensor問い合わせるセンサーオブジェクト
float *dataセンサーの状態の書き込み先へのポインタ
intnum_valuesdataに書き込まれた値の数

戻り値

(bool) 成功のとき真, 失敗のとき偽を戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

値の数とデータの意味はセンサーに依存する.

バージョン

SDL 3.2.0以降
